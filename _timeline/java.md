caption: Τον κύριο και βασικό ρόλο στην ανάπτυξη της java είχε ο James Gosling, που εργαζόταν εκείνη την εποχή στην Sun. Βασικό πλεονέκτημα της συγκεκριμένης γλώσσας είναι η ανεξαρτησία που διαθέτει το λειτουργικό σύστημά της, καθώς και η πλατφόρμα της. Οι κώδικες σε java δεν διαφέρουν σε Windows, Linux, Unix και Macintosh. Επίσης, γίνονται πειράματα και εργασίες για να μπορούν να τρέχουν οι κώδικες σε Playstation και άλλες κονσόλες παιχνιδιών. Αυτό θέλουν να το πετύχουν με τρόπο τέτοιο, ώστε να μην χρειάζονται αλλαγές για κάθε σύστημα, λόγω του διαφορετικού επεξεργαστή του και του λειτουργικού, του συστήματος.
